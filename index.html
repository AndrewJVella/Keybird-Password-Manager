What is Keybird?

The purpose of Keybird is to improve the accessibility of password managers
without compromising security. It generates and store passwords on your computer.

You need a lot of passwords but making them is tedious.
In-browser managers can lock you out.

You may ask yourself:
"What was my password again?"
"Why does Safari fill in the wrong info?"
"Did I make this account on Google?"
Skip that nonsense, use Keybird. Generate secure passwords from simple keys,
which are easy to create and remember.




Features

-A One Key password generator
-A Two Key password generator, with a public and private key
-A Username Generator
-A Notepad with accident preventing encryption
-Local Password Storage, with file editing and accident preventing encryption
-Password Masking to prevent over-the-shoulder attacks
