Keybird Password Manager Web Prototype


This is a DOS like prototype for an app to facilitate password management. You can generate passwords that are secure with keys that are easy to rememeber.

This is the Trinket prototype, it may take a moment to load. Please be sure to click the "run" button.

On Github

The hope is that this app will replace two common, bad, practices. Manuallly generating passwords is tedious, and the passwords are hard to remember. Browsers have automatic solutions that seem fine until they lock you out. This is a hybrid solution. You can use it with your password manager if you like.

Known Issues (In Order of Priority)

Vunerability to over-the-shoulder attacks is present. (People behind you can see what you are typing here, which is not secure.) Coloring the text to a background color is useful for solving this issue.
