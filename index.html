What is Keybird?

The purpose of Keybird is to improve the accessibility of password managers without compromising security. It generates secure passwords from simple keys, which are easy to create and remember. These can be stored on your computer.

You need a lot of passwords but making them is tedious and in-browser managers can lock you out. You may ask yourself:

"What was my password again?"
"Why does Safari fill in the wrong info?"
"Did I make this account on Google?"

Skip that nonsense; use Keybird.


Features

-A One Key password generator
-A Two Key password generator, with a public and private key
-A Username Generator
-Local Password Storage, with File Editing
-A File Organizer that optimizes storage
-Password Masking to prevent over-the-shoulder attacks
-Accident Preventing Encryption
-A Notepad
-Accessibility: flashing light prevention; selections with home row.
-Adjustable Settings
